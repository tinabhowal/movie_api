myFlix

myFlix is a REST API for an application called “myFlix” that interacts with a database that stores data about different movies.


Essential Features

  • Return a list of ALL movies to the user.
  • Return data (description, genre, director, image URL, whether it is featured or not) about a single movie by title to the user.
  • Return data about a genre (description) by name/title (e.g., “Thriller”) and movies of that genre.
  • Return data about a director and movies directed by him/her. (bio, birth year, death year) by name.
  • Return data about movies with the requested actor.
  • Allow new users to register.
  • Allow users to update their user info (username, password, email, date of birth).
  • Allow users to add a movie to their list of favorites.
  • Allow users to remove a movie from their list of favorites.
  • Allow existing users to deregister.

API Endpoints

Buisness Logic URL HTTP Method Query Parameters Request body data format Response body data format
Get a list of all movies /movies GET - - A JSON object holding data about all the movies.
Get a specific movie by title /movies/:title GET :title - A JSON object holding data about a single movie.
Get movie by genre name /movies/genre/:genreName GET :genreName - A JSON object holding data about the movies of the requested genre.
Get movies by a particular director /movies/director/:directorName GET :directorName - A JSON object holding data about movies by the requested director.
Get movies having a particular actor /movies/actor/:actorName GET :actorName - A JSON object holding data about movies having the requested actor.
